# Arquitetura e Desenho de Software - Programa de Gestão para Profissionais de Saúde Mental (PSIs)

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de uma plataforma **Software as a Service (SaaS)** dedicada à gestão clínica, administrativa e financeira para profissionais de saúde mental no Brasil. O sistema visa mitigar a carga de trabalho dos profissionais diante da alta demanda do cenário demográfico brasileiro e garantir a conformidade com as rigorosas exigências éticas e legais do setor.

## 🎯 Escopo do Sistema
O sistema é focado em resolver o "baseline" operacional do consultório (Necessidades Dorsais) e oferecer diferenciais competitivos.
